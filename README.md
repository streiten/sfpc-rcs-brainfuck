### SFPC: Radical Computer Science - Brainfuck

Expirience a true classic in esoteric languages!
The following beautiful pattern

```
++++[>+++++[
>++>+++<<-]<
-]>>-------.
+++++++.>++.
<++++.>--.<-
--.--------.
```

generates this emoticon
```
!(>,<)!
```
